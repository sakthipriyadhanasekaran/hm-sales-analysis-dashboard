# H&M Sales & Supply Chain Analysis Dashboard

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze H&M sales performance, customer behavior, inventory trends, logistics operations, and financial profitability.

The dashboard helps identify key business insights and supports data-driven decision-making.

---

## Dashboard Features
- Executive KPI Overview
- Customer Segmentation Analysis
- Product & Inventory Insights
- Supply Chain & Logistics Tracking
- Financial & Profitability Analysis
- Regional Sales Performance

---

## Tools & Technologies Used
- Power BI
- Excel / CSV Dataset
- DAX
- Data Cleaning
- Data Visualization

---

## Key Metrics
| Metric | Value |
|--------|-------|
| Total Sales | 36.78M |
| Total Profit | 3.97M |
| Total Orders | 180K+ |
| Avg Delivery Time | 3.5 Days |

---

## Dashboard Pages

### 1. Executive Overview
Provides a summary of:
- Sales
- Profit
- Orders
- Delivery performance
- Regional analysis

### 2. Customer Analysis
Focuses on:
- Customer segments
- Repeat customers
- Sales behavior
- Customer trends

### 3. Product & Inventory Analysis
Analyzes:
- Top-selling products
- Product categories
- Inventory value
- Quantity sold

### 4. Supply Chain & Logistics
Tracks:
- Delivery status
- Shipping modes
- Late deliveries
- Regional logistics performance

### 5. Financial Analysis
Includes:
- Profit trends
- Cost analysis
- Profit margin by category
- Monthly financial performance

---

## Key Business Insights
- Consumer customers generated the highest sales revenue.
- Camping & Hiking category showed strong profitability.
- Late delivery percentage was significantly high.
- Standard Class shipping mode handled the majority of orders.
- Certain regions demonstrated stronger profit margins than others.

---

## Business Recommendations
- Improve logistics tracking in high-delay regions.
- Optimize shipping processes to reduce late deliveries.
- Focus marketing efforts on high-performing product categories.
- Improve inventory planning using demand trends.
